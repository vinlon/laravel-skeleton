# Laravel Skelton 应用开发基础模板

Based On: https://github.com/laravel/laravel/tree/7.x

## 功能：

- 增加 DB_PREFIX 环境变量
- 引入 SQL-LOGGER
- 引入 IDE-HELPER并调整配置
- 配置PHP-CS-FIXER
- .env.example 简化 （只保留必要的项）
- 增加 scratch.php 文件


## 使用 

### 使用composer create-project 创建项目

```shell script
composer create-project --prefer-dist laravel/laravel:^7.0 blog
```
